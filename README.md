## Instalasi

```bash
1. git clone https://github.com/StarTop-Indonesia/kriptografi-api.git
2. cd kriptografi-api
3. pip install virtualenv
4. virtualenv myenv
5. source myenv/scripts/activate
6. pip install flask
7. pip install flask-restful
8. pip install flask-cors
9. pip install gunicorn
```

## Usage

```bash
1. Masuk ke folder kriptografi-api
2. source myenv/scripts/activate
3. python app.py
```
